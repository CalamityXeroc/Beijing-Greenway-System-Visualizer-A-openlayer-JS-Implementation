# 安全与开源政策 (Security & Open Source Policy)

## 🔐 安全信息

### 敏感信息保护

本项目遵循以下安全实践：

1. **不存储敏感信息**
   - API密钥、数据库密码等敏感信息仅存储在 `.env.local` 文件中
   - `.env.local` 文件已添加到 `.gitignore` 中，永远不会提交到 Git

2. **环境变量配置**
   - 所有配置应通过环境变量 (`.env`) 管理
   - 源代码中不应包含硬编码的密钥或密码
   - 使用 `.env.example` 作为配置模板

3. **API密钥管理**
   - 高德地图 API Key：申请地址 https://lbs.amap.com/
   - 百度地图 API Key：申请地址 https://lbsyun.baidu.com/
   - 每位开发者应自行申请和配置

### 数据库安全

- 默认密码 `postgres` 仅供本地开发使用
- 生产环境必须使用强密码
- 建议使用连接池和参数化查询防止 SQL 注入

### 前端安全

- 所有 API 调用使用 HTTPS（生产环境）
- CORS 已配置，仅允许受信任的源
- 用户输入已进行验证

## 🔒 关于开源与知识产权

### 项目状态

本项目为**部分开源**项目，遵循以下政策：

#### 已公开内容
- ✅ 前端 Vue 3 + OpenLayers 实现代码
- ✅ 后端 Express API 服务代码
- ✅ 数据库架构和 GeoJSON 数据
- ✅ 项目文档和使用指南
- ✅ 基础脚本和工具

#### 暂不公开内容（为学术出版和竞赛保护）
- ❌ 完整的高德地理数据集（其他绿道数据）
- ❌ 高级空间分析算法和优化算法
- ❌ 竞赛解决方案代码
- ❌ 论文未发表的创新内容

### 原因说明

本项目正在进行以下学术活动：

1. **学术竞赛参赛**
   - 参加各类 GIS 和地理信息技术竞赛
   - 为保持竞争优势，核心算法未公开

2. **论文发表**
   - 研究成果计划投稿至学术期刊
   - 论文发表前需保持原创性和保密性

3. **软件著作权申请**
   - 已/计划申请中国软件著作权
   - 保护创意实现和核心代码
