---

### 问题2: IP访问失败（防火墙）
**症状**:
- ✅ curl http://localhost:3001/health 成功
- ❌ curl http://10.26.171.159:3001/health 失败

**解决**: 添加防火墙规则
```cmd
# 以管理员身份运行
netsh advfirewall firewall add rule name="Greenway Backend" dir=in action=allow protocol=TCP localport=3001
```

---

### 问题3: 应用无法加载数据
**症状**: 应用打开但显示空白或无数据

**解决方案A**: 完全重新安装
```cmd
cd greenway-vue
adb uninstall com.greenway.app
npx cap run android
```

**解决方案B**: 使用应用内诊断
1. 打开应用
2. 进入"设置" → "网络诊断"
3. 点击"🔄 重新测试所有"
4. 查看详细日志

## 💡 重要提示

### 关于IP地址
- ✅ **正确**: `10.26.171.159` (WLAN地址)
- ❌ **错误**: `169.254.127.251` (自动配置地址)
- ℹ️ IP可能因网络变化而改变

### 关于防火墙
- Windows防火墙可能阻止外部访问
- 需要为3001端口添加入站规则
- 临时方案：关闭防火墙测试

### 关于缓存
- 热重载可能不生效
- 建议完全卸载后重新安装
- 应用内可以清除缓存

### 关于VPN
- VPN/加速器会改变网络配置
- 调试时建议关闭
- 如使用VPN，IP地址会变化
